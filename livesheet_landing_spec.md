# Project: LiveSheet (라이브시트) Landing Page Spec

## 1. Project Overview
- **Service Name:** LiveSheet (라이브시트)
- **Target Audience:** Academy Directors (Principals) via "Gongboo Senior".
- **Goal:** Persuade directors to adopt the service by highlighting "Cost Efficiency," "Asset Protection," "Extreme Customization," and "Pricing Merit."
- **Key Tone:** Professional (Navy), Premium/Bespoke (Gold), High-Tech (Orange).

## 2. Design System
- **Colors:**
  - Primary: Navy Blue (#0F172A) - Trust.
  - Accent: Vivid Orange (#F97316) - CTA & Key numbers.
  - Highlight: Soft Gold (#D4AF37) - Premium Customization.
- **Typography:** Pretendard or Noto Sans KR (Clean Sans-serif).

## 3. Content Structure (Section by Section)

### Section 1: Header
- **Logo:** LiveSheet
- **Badge:** "공부선배 공식 파트너 솔루션"

### Section 2: Hero Section (Impact)
- **Headline:** "학원 운영, 시스템에 맞추지 마세요. 시스템을 학원에 맞추세요."
- **Sub-headline:** "매주 기능이 업데이트되는 '나만의 문제은행'. 원장님은 요청만 하세요. 나머지는 저희가 합니다."
- **CTA Button:** [우리 학원 맞춤 상담 신청하기]
- **Visual:** A puzzle piece fitting perfectly into an academy's workflow.

### Section 3: The "Tailoring" Promise (Killer Feature)
- **Title:** "프로그램이 아닙니다. '나만의 개발팀'입니다."
- **Subtitle:** "기성복 같은 프로그램은 이제 그만. 1주일 안에 필요한 기능을 만들어 드립니다."
- **Feature Card 1 (Speed):** "필요한 기능 말씀만 하세요. 1주일 내 개발 및 탑재 완료."
- **Feature Card 2 (Workflow):** "로그인 화면부터 배포 동선까지, 학원 프로세스에 100% 맞춤 설계."

### Section 4: Device Flexibility (Objection Handling)
- **Title:** "태블릿? 없어도 됩니다. 기기 걱정 ZERO."
- **Grid:**
  1. **URL 배포:** 파일 업로드 -> 학생 카톡 링크 전송 (1초 배포).
  2. **All Devices:** 스마트폰, PC, 노트북, 태블릿 완벽 호환.
  3. **Hybrid Mode:** 종이가 편한 학생을 위한 워터마크 PDF 출력 지원.

### Section 5: Comparison (Efficiency)
- **Title:** "핸드아웃 vs 라이브시트"
- **Table:**
  | 구분 | 기존 핸드아웃 | 라이브시트 |
  | :--- | :--- | :--- |
  | 비용 | 종이/토너값 지속 발생 | **비용 0원 + 자산화** |
  | 채점 | 수기 채점 (시간 낭비) | **AI 자동 채점 & 리포트** |
  | 커스텀 | 불가능 (주는 대로 사용) | **필요한 기능 맞춤 개발** |
  | 관리 | 회수 불가, 분실 빈번 | **즉시 회수, 열람 추적** |

### Section 6: Pricing & Value (ROI Anchor)
- **Title:** "월 7,000원으로 전문 개발팀을 고용하세요."
- **Subtitle:** "개발자 채용, 서버 운영, 유지보수... 골치 아픈 건 저희가 합니다."
- **Comparison Visual:**
  - **A. 자체 개발팀 구축:** 월 500만 원++ (채용 스트레스, 고정비 부담) 😱
  - **B. 라이브시트 도입:** **학생당 월 7,000원 (커피 한 잔 값 / 쓴 만큼만 결제)** 😎
- **Message:** "공부선배 프로모션 기간에는 심지어 무료로 시작할 수 있습니다."

### Section 7: Final CTA (Risk Reversal)
- **Title:** "고민은 써보고 하셔도 늦지 않습니다."
- **Subtitle:** "공부선배 원장님 한정, 리스크 없는 3가지 혜택을 드립니다."
- **Benefit List:**
  1. **가입비/초기 세팅비 전액 면제 (0원)**
  2. **Premium Plan 1개월 무료 체험 (결제 정보 입력 X)**
  3. **[선착순] 자체 교재 문제은행 구축 지원 & 커스텀 개발권**
- **Main Action:** [무료로 내 개발팀 만들기]
- **Footer Note:** "도입 문의 시 학원 운영 프로세스에 맞춘 최적의 동선을 설계해드립니다."

## 4. Technical Requirements
- **Framework:** React, Tailwind CSS.
- **Responsiveness:** Mobile-first design (Must look like a high-end app flyer on mobile).
- **Export:** Designed to be clean enough to screenshot as a single-page PDF.
